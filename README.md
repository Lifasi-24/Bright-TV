 <h1>📺BrightTV Viewership Analytics Case Study<h1>

<h2>🗒️Project Overview</h2>
<p>This project explores BrightTV's subscriber viewership data to gain insights into customer behavior, analyze viewing patterns, and identify the key factors influencing content consumption. The findings are used to develop data-driven recommendations that support the Customer Value Management (CVM) team in improving user engagement, increasing viewer retention, and expanding BrightTV's subscriber base.

The analysis follows a comprehensive end-to-end data analytics process, encompassing project planning, data preparation and cleaning, SQL-based analysis in Databricks, interactive dashboard creation, data visualization, and the delivery of actionable business insights and strategic recommendations.</p>



<h2>📖Business Objective</h2>
<p>BrightTV's CEO <strong>aims to increase the company's subscription base</strong> during the current financial year.

Using customer profile and viewing session data, this project answers key business questions such as:

* What are the current user and viewing trends?
* Which factors influence content consumption?
* What content should be promoted on low-consumption days?
* Which initiatives could increase subscriber growth and retention?</p>

<h3>❗Key Insights</h3>
<p>Examples of insights explored include:

* User engagement patterns
* Peak viewing hours and days
* Most popular content categories
* Viewing frequency
* High-value customer segments
* Factors influencing content consumption</p>
